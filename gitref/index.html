Git Reference

Commands

init
Create a new git repository for a project.
add || ga
Adds files to the stage.
status || gs
Information about files to be committed, and what step along the commit process you are currently on.
log || glg
Shows the sequence of commits in the repository.
commit
Saves a snapshot of the contents of the stage, then clears the stage.

Don't forget -m

How to quit Vim: [Esc]:wq!

--amend
Modifies the previous commit
gitstup

Usage: git setup Zenoni zenonira@s.dcsdk12.org

Runs init and sets user name and emial.

reset

Usage" git reset HEAD [file]

Unstages a staged file.

Git Terms

Repository
A Collection of snapshots of the progress of a project through time.
Stage
A temporary area for holdig files to be committed.
Snapshot
A version of the code that can be returened to somewhere easily.
Head
The current snapshot(the previouse commit)
